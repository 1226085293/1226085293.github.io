      • 功能简介

      在经过了几年的 cocos 游戏开发开发生涯后，自我沉淀促使我开始编写这个 cocos creator 3.x 游戏框架。

      对于做游戏还是做工具，我发现自己更喜欢做工具

      注意

      目前此框架处于 beta 版本，因为自己一直没花时间去做一个完整的游戏，所以我不想让用户承担风险

      # 功能简介

      音频系统
      (微信/通用)双版本系统，支持动态加载音频，批量控制音频单元，内置音频事件
      引导系统
      多边形(触摸/遮罩)组件，多实例，步骤跳转，步骤通信，解决跳转任意步骤后步骤的状态管理
      多语言系统
      模块式多语言配置（避免垃圾堆积），多语言图片，多语言文本，多语言状态节点
      模块系统
      UI 栈，UI 池，动态加载，全屏优化，生命周期管理
      网络系统
      (微信/通用)双版本系统，多实例，消息潮，网络事件化，protobuf 使用简化
      资源系统
      bundle 管理器，bundle 热更机制，强制资源管理，简化加载接口，提供资源生命时长，解决引擎bug
      其他
      UI 适配，(同步/异步)对象池，存储器，数据共享器，数据监听器，类型安全事件对象...
