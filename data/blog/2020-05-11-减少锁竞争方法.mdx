设置双重进入锁判断，例：

```cpp
bool enter = false;

void test() {
    if (enter) {
        return;
    }
    mutex.lock();
    if (enter) {
        return;
    }
    enter = true;
    mutex.unlock();
    // 代码块..
    enter = false;
}
```
