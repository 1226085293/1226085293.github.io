1. 项目根目录 `npm i ts-node`

1. launch.json 内 configurations 添加如下内容
   ```json
   {
     "name": "当前 ts 文件",
     "type": "node",
     "request": "launch",
     "program": "${workspaceRoot}/node_modules/ts-node/dist/bin.js",
     "args": ["${relativeFile}"],
     "cwd": "${workspaceRoot}",
     "protocol": "inspector"
   }
   ```
