数组示例

```ts
let users = [ {"name":"张三","age": 10},
{"name":"李四",age: 14},
{"name":"王五",age: 9},
...
]
```

要求统计出数组里年龄大于 10 的人数。

方法一：使用 filter 和 length

```ts
const result = users.filter((u) => u.aage > 10).length
```

方法二：使用 reduce

```ts
const result = users.reduce((c, u) => u + (u.age > 10), 0)
```
