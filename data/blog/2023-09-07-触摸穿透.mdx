  - CocosCreator
---

- 3.4 之前：

```typescript
this.node.eventProcessor.touchListener.setSwallowTouches(false)
```

- 3.4 及之后

```typescript
// event_ 类型: cc.EventTouch | cc.EventMouse

// 触摸穿透
event_.preventSwallow = true
// 拦截事件
event_.propagationStopped = false
```
