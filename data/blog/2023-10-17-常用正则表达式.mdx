## 常用正则

### 匹配除x外所有字符

```
([^x]*)
```

### 匹配指定前后缀内容

```
(?<=前缀)([^]*)(?=后缀)
```

### 不能以\_开头

```
(?!_)
```

### 不能以ab结尾

```
(.*(?<!ab)$)
```

### 前面不存在abc

```
(?<!abc)([^]+)
```

---

## 示例

- 只含有汉字、数字、字母、下划线不能以下划线开头和结尾：

```
(?!_)(?!.*?_$)[\w\d_u4e00-u9fa5]+
```
