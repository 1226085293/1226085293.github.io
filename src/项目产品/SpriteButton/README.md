还在苦恼不规则按钮点击？苦苦的拖动编辑多边形点？快来试试这个组件吧，通过检测像素触发事件，让你再也不必为了奇形怪状的按钮苦恼

<!-- more -->

## 效果图

![](https://gitee.com/muzzik/images/raw/master/%E9%A1%B9%E7%9B%AE%E4%BA%A7%E5%93%81/SpriteButton/1.gif)

## 功能介绍

通过点击位置像素透明度判断是否触发事件，可同于多边形/不规则按钮，比如大陆板块，镂空按钮等等...

## 支持特性

- 旋转

- 缩放
- 大小调整
- 支持 plist / 动态合图 / 自动图集 / 散图
- sprite trim 模式支持
- 按钮叠加
