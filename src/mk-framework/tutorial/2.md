MKFramework 推荐你把自己的想法加入框架代码，所以直接将代码放在了 assets/@framework 下，方便你的调试和修改，如果你有什么好的代码或者想法，建议提交到框架仓库

<!-- more -->

如果你不知道怎么提交代码，可在企鹅交流群询问 200351945

## 生成框架 d.ts

当你修改框架代码内的接口后，你会发现 `mk.xxx` 的接口还是没有变化，这时候你需要重新生成框架的 d.ts 文件，只需要在项目根目录下终端执行 `npm run "构建框架 d.ts"` 即可
