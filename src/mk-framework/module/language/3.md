多语言节点是为了满足多语言图片和文本不能满足的场景而存在，例如节点位置，缩放，或其他参数的不同

<!-- more -->

## 使用

添加 mk_language_node 组件到两个不同语言展示节点的父节点（不限），选择语言后拖动对应节点赋值即可

![](https://gitee.com/muzzik/images/raw/master/mk-framework/module/language/3/1.png)

- 多语言节点增加了 layout 节点的适配，所以你也可以单独对 Layout 容器节点使用
