## 依赖环境

- Nodejs

## 安装步骤

1. 通过 Cocos 商店添加到项目，链接：https://store.cocos.com/app/detail/6426

1. 关闭编辑器，打开插件目录 项目根目录\extensions\MKFramework 后在当前目录执行命令
   ```bat
   npm install
   ```
   等待执行成功打开编辑器

1. 打开编辑器，点击顶部菜单/扩展/mk-framework/安装框架  

1. 等待安装成功后重启编辑器即可使用，有任何问题欢迎提出
