1. 下载 mk-install.zip
> <a href="https://github.com/1226085293/MKFramework/wiki/%E5%AE%89%E8%A3%85%E7%A8%8B%E5%BA%8F" target="_blank">点击此处在新标签页中打开下载链接</a>
 
2. 在项目根目录下 extensions 文件夹解压

3. cd 到 `extensions/mk-install` 目录下，命令行 `npm i` 安装依赖包
> 如果 `npm i` 中断或报错，需删除 `node_modules` 文件夹后重新执行 `npm i`
> 成功安装截图：
> ![](https://gitee.com/muzzik/images/raw/master/mk-framework/quick-start/1/Z\(LQE4~U5%7D%7D~LM3YM@AA1X7.png)

4. 打开 cocos creator 扩展管理器启用 mk-install 插件

1. 按下 f3 键唤出安装面板
