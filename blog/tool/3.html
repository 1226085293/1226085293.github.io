                                    # # 介绍

                                    开发版本 3.3.2，支持 任意动画曲线

                                    • 滚动抽奖
                                      • 支持无限循环滚动
                                      • 支持 横 / 竖 滚动
                                      • 实时更新当前选中节点（父节点（0,0）坐标）
                                      • 可选自定义贝塞尔曲线动画
                                      • 可选内置缓动函数
                                    • 旋转抽奖
                                      • 支持顺时针、逆时针旋转
                                      • 支持转盘、指针旋转
                                      • 实时更新当前选中节点
                                      • 可选自定义贝塞尔曲线动画
                                      • 可选内置缓动函数
